Angular Demo

This AngularJS demo makes use of AngularJS combined with a controller. To collect the user input, ng-model is used to bind input to data. Expressions are used to output the data to the HTML view in the resulting paragraph. The controller includes a hide function through which the paragraph can be hidden until the user input is submitted.
