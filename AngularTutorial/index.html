AngularJS

What is AngularJS?

AngularJS is a JavaScript framework developed by Google for building well-structured dynamic web applications. It extends HTML syntax using directives and allows you to bind JavaScript data to HTML. Angular eliminates the need for writing extensive code, therefore making way for easy-to-build and easy-to-maintain applications.

AngularJS Unique Qualities

  • Allows for easy building of single-page applications
  • Extends HTML with attributes such as directives and data binding, modules, controllers
  • Eliminates the need for direct DOM manipulation
  • Applications are easy to test within the browser

Getting Started with AngularJS

To get started with AngularJS, you will need to include the Angular scripts within your HTML document. This can be done in one of two ways: including the link from the Google CDN server or downloading the files for a local copy.

Hosting from Google CDN Server

The Angular script should be included within the head of your document. You may include either the minified or non-minified version. In this case, minified version 1.4.8 is included.

Hosting Locally

Head to http://code.angularjs.org/ to select a version of Angular to download from the directory. To begin, version 1.4.8 may be a good choice as used in the last example.

Angular Syntax and Examples

Here is an example of a basic AngularJS application. You will notice extended HTML attributes which allow for data binding with JavaScript.

Extended HTML, Explained

  • ng-directives: Angular directives are defined with the prefix ng-
  • ng-app: ng-app defines an Angular application
  • ng-model: ng-model binds HTML values to data
  • ng-bind: ng-bind binds the data values to innerHTML
  • expressions: Angular expressions may display data directly where the expression is written. Expressions are written in the form of {{ expression }}. They may contain data types such as model names or number expressions.

The message input field was extended with the ng-model "message" which binds the input text to JavaScript data. ng-bind "message" binds the collected data to the innerHTML of the paragraph. Expressions perform the same actions as the ng-bind directives, therefore writing a duplicated message underneath the first.

Modules and Controllers

Angular modules and controllers define the JavaScript side of your application.

  • module: The module defines the application and contains its contents. It always contains a controller. What the module is defined as should be used in the ng-app definition within the HTML.
  • controller: An Angular controller is a JavaScript object which defines the data necessary for your application. In this example, the directive "myCtrl" defines the controller. $scope is a default object that will contain the data.

Other AngularJS Capabilites

While this page reviews the basics of working with simple AngularJS applications, AngularJS has many other capabilities that ease the development of dynamic web apps. Some of these capabilities include:

  • Reading data from web servers using $http
  • Easing table or list creation with ng-repeat
  • Reading from SQL databases
  • Contains unique event functions
  • Can be easily integrated with Bootstrap
